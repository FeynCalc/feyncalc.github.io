Hi Cesar,  

C0 functions have certain symmetries in their arguments. Those  
are applied by PaVeReduce:  

f1 = PaVe[0, 0, {0, 0, m1^2}, {0, m2^2, m2^2}] // PaVeReduce  
f2 = (PaVe[0, 0, {m0^2, 0, m1^2}, {0, m2^2, m2^2}] //
PaVeReduce) /.  
  m0 -\> 0  
PaVeOrder[f1 - f2]  

returns 0.  

Cheers,  
Vladyslav
