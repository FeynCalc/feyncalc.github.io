FEYN CALC SYMBOL

Tensorfunction


transform into t[LorentzIndex[mu], LorentzIndex[nu], ...], i.e., it can be used as an unspecified tensoriell function t. A symmetric tensor can be obtained by Tensorfunction[{t, "S"}, mu, nu, ...], and an antisymmteric one by Tensorfunction[{t, "A"}, mu, nu, ...].
