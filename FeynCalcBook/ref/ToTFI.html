ToTFI

ToTFI[expr, q1, q2, p]
translates FeynCalc 2-loop self energy type integrals into the TFI notation, which can be used as input for the function TarcerRecurse from the TARCER package. See the TARCER documenatation on TFI for details on the conventions.
