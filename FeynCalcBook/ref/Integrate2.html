Integrate2

Integrate2
is like Integrate, but : Integrate2[a_Plus, b__] := Map[Integrate2[#, b]&, a] ( more linear algebra and partial fraction decomposition is done) Integrate2[f[x] DeltaFunction[x], x]  f[0] Integrate2[f[x] DeltaFunction[x0-x], x]  f[x0]. Integrate2[f[x] DeltaFunction[a + b x], x]  Integrate[f[x] (1/Abs[b]) DeltaFunction[a/b + x], x], where abs[b]  b, if b is a Symbol, and if b = -c, then abs[-c]  c, i.e., the variable contained in b is supposed to be positive. is replaced by 6 Zeta2. Integrate2[1/(1-y),{y,x,1}] is intepreted as distribution, i.e. as Integrate2[-1/(1-y)],{y, 0, x}]  Log[1-y]. Integrate2[1/(1-x),{x,0,1}]  0.

DetailsDetails

  • Since Integrate2 does do a reordering and partial fraction decomposition before calling the integral table of Integrate3 it will in general be slower compared to Integrate3 for sums of integrals. I.e., if the integrand has already an expanded form and if partial fraction decomposition is not necessary it is more effective to use Integrate3.
