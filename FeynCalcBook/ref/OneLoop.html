OneLoop

OneLoop[q, amplitude]
calculates the one-loop Feynman diagram amplitude. The argument q denotes the integration variable, i.e., the loop momentum. OneLoop[name, q, amplitude] has as first argument a name of the amplitude. If the second argument has head FeynAmp then OneLoop[q, FeynAmp[name, k, expr]] and OneLoop[FeynAmp[name, k, expr]] tranform to OneLoop[name, k, expr].
