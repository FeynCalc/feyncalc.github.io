FCRenameDummyIndices

FCRenameDummyIndices[expr]
identifies all dummy Lorentz and SU(N) indices and changes their names pairwise to random symbols. This can be useful if you have an expression that contains dummy indices and want to compute the square of it. For example, the square of GA[a, l, a] equals 16. However, if you forget to rename the dummy indices and compute GA[a, l, a, a, l, a] instead of GA[a, l, a, b, l, b], you will get 64.
