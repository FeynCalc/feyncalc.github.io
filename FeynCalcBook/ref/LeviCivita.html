LeviCivita

LeviCivita[mu,nu,rho,si]
is an input function for the totally antisymmetric Levi-Civita tensor. It evaluates automatically to the internal representation Eps[LorentzIndex[mu], LorentzIndex[nu], LorentzIndex[rho], LorentzIndex[si]] (or with a second argument in LorentzIndex for the Dimension, if the option Dimension of LeviCivita is changed). LeviCivita[][ .] evaluates to Eps[LorentzIndex[], LorentzIndex[], ..., Momentum[], ...].
