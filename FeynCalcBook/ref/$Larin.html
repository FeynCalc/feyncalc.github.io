FEYN CALC SYMBOL

$Larin


If set to True, the Larin-Gorishny-Atkyampo-DelBurgo-scheme for in D-dimensions is used, i.e. before evaluating traces (but after moving anticommuting in all dimensions to the right of the Dirac string) a product gamma[mu].gamma5 is substituted to -I/6 Eps[mu,al,be,si] gamma[al,be,si], where all indices live in D-dimensions now. Especially the Levic-Civita tensor is taken to be D-dimensional, i.e., contraction of two Eps's results in D's. This has (FOR ONE AXIAL-VECTOR-CURRENT ONLY, it is not so clear if this scheme also works for more than one fermion line involving ) the same effect as the Breitenlohner-Maison-'t Hooft-Veltman scheme.
