$LeviCivitaSign

$LeviCivitaSign
is a global variable that determines the sign in the result of a Dirac trace of four gamma matrices and gamma5. $LeviCivitaSign is by default set to -1 which corresponds to the convention Tr[LC[a,b,c,d,5]] = -4*I*Eps[a,b,c,d]. Setting $LeviCivitaSign=-I will switch to the FORM-convention.
